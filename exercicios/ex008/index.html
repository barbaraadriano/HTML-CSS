Principais formatações

Negrito / destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica)

Nesta frase temos um termo em destaque usando a tag STRONG (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico (não semântica)

Nesta frase, temos um termo em ênfase usando a tag EM (semântica)

Não que usar a tag B e a tag I seja errado, a W3C não disse nada sobre ainda, mas como HTML5 fala sobre semântica, é melhor usar a tag STRONG para negrito e a tag EM para itálico

Texto marcado

Podemos também criar um texto marcado usando a tag MARK

E no outro parágrafo a cor do MARK não fica igual a do primeiro, se quisermos isso, temos que configurar no HEAD do código (configurei)
